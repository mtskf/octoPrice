## Setup

#### Install dependencies
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn start
```

#### Compiles and minifies for production
```
yarn build
```


## Requirements

 * [node v12+](https://nodejs.org/en/download/)
 (Developed with Node v16.15.0)
